- 인스타그램을 모티브로 한 소셜네트워크 웹페이지 구현
- React를 사용한 SPA 웹페이지 구현 프로젝트
- 팀원들 각자가 구현한 동일한 웹페이지를 하나의 프로젝트로서 GitHub repository 구성
